// THE PROBLEM

Building the agent is easy. Trusting it with real access is not.

An always-on agent with live credentials and a data store can be talked into crossing a line it should never cross. A policy written once is only as good as the attacks nobody tried. Red/Blue Arena turns that guesswork into a measured, co-evolving loop.

01 · boundaries are untested

Nobody knows where a policy leaks until an adversary probes it under pressure. So let one probe it, every round.

02 · detection is passive

Most guards refuse and stop. Ours feeds the finding back so blue closes that whole class of attack.

03 · improvement must compound

Real recursion needs a measurable delta between the first run and the last — not a cache dressed up as learning.

04 · security can't live in goodwill

A boundary in the system prompt walks through on the first injection. It has to live in the OpenShell runtime.

// THE PIPELINE

Six stages. One round of co-evolution.

Red attacks, the runtime detects, the controller diagnoses, blue hardens, the loop verifies, the harness measures. Repeat until the rate flattens near zero.

01 · ATTACK
red-team

Craft exfil

Nemotron proposes a battery of strategies to move seeded secrets past the current policy.

02 · DETECT
HiddenLayer

Flag at runtime

All five target I/O surfaces route through the Runtime Security API. Fail-closed on error.

03 · DIAGNOSE
controller

Find the hole

Read HiddenLayer and OpenShell traces; identify which attempt landed and how.

04 · HARDEN
blue-team

Rewrite policy

Nemotron rewrites openshell-policy.yaml to close the hole without breaking approved egress.

05 · VERIFY
sandbox

Rerun

Patch parses, approved calls still pass, the same attack is now denied. Promote the policy.

06 · MEASURE
R2

Record delta

Snapshot the round's policy, traces, and exfil-success-rate. The curve steps down.

// TECHNOLOGY STACK

Every sponsor tech is load-bearing.

Pull any one out and the loop stops working. Nothing here is a token mention.

OpenShell · the boundary

YAML policy is the sole guard on target→sink. Docker egress stays open on purpose so the policy is the only thing holding the line.

HiddenLayer · the sensor

Five I/O surfaces instrumented, fail-closed. Findings drive the loop, not just a refusal.

Nemotron + vLLM · the brain

One self-hosted endpoint on Brev serves both sides — a real concurrent batching workload under the heartbeat.

NemoClaw · the harness

Runs the target inside the OpenShell sandbox with routed inference. The reference stack for always-on agents.

Cloudflare · the memory

Worker Assets serve these docs; D1 logs reasoning, KV holds round state, R2 snapshots every round.

Docker Compose · the arena

control-net isolates the plane we're not judged on; data-net keeps egress open so OpenShell does the security.
