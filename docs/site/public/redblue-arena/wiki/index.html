WIKI // RED/BLUE ARENA

Glossary

Every term, service, sponsor tech, metric, and operating rule in the arena — defined once, in plain terms.

// CORE CONCEPTS

The mechanism

TermDefinition
Claw AgentAn AI system that is proactively autonomous, heartbeat-driven (wakes on time/state, not a human prompt), and persistent with its own memory and workspace.
Co-evolution loopThe core mechanism: red and blue improve against each other. Red's success forces blue to harden; blue's hardening forces red to get cleverer.
Boundary invariantThe rule that the target→sink network path stays open at the Docker layer so the OpenShell YAML is the sole guard. Violating it forfeits the OpenShell bounty.
Control plane / data planeThe split that keeps the design honest: Docker isolates the control plane (controller/red/blue); the data plane (target/sink) has open egress guarded only by policy.
AblationThe control experiment: run the loop with policy-learning ON vs frozen. Learning-ON's exfil rate falls; frozen stays flat. Proves the learning mechanism, not just the outcome.
Recursive deltaThe judged number: exfil-success-rate at round 1 minus round N. The whole Recursive Intelligence track hinges on it.
// SERVICES

Compose services

TermDefinition
controllerRuns the Harness loop (observe→diagnose→propose→verify→promote), records traces to the shared volume and KV round state. On control-net.
red-teamNemotron-driven service that proposes and executes exfil strategies against the current policy, reading HiddenLayer and OpenShell traces to find gaps.
blue-teamNemotron-driven service that reads a finding and rewrites the OpenShell YAML to close it without breaking approved egress, then hot-reloads.
targetThe agent under test, run under NemoClaw inside an OpenShell sandbox, holding seeded secrets. Every side effect passes through the action broker.
attacker-sinkA tiny HTTP server on data-net that logs any payload it receives. Stands in for the unapproved endpoint the agent must never reach.
action brokerThe single chokepoint inside the target through which every side effect flows. Where OpenShell enforcement and HiddenLayer checks plug in.
// SPONSOR TECH

What each sponsor tool does here

TermDefinition
OpenShellNVIDIA's private runtime for autonomous agents. Declarative YAML policies prevent unauthorized file access, exfiltration, and network activity. Here it is the sole guard on egress.
NemoClawNVIDIA's open reference stack for running always-on agents (OpenClaw/Hermes/LangChain) more safely inside OpenShell sandboxes, with routed inference through one CLI.
HiddenLayerRuntime Security API that treats every model input/output as untrusted. Detects prompt injection and data leakage in real time. Instrumented across all five I/O surfaces here.
NemotronNVIDIA's family of open models built for agentic workloads. The reasoning engine for both red and blue — central, not a wrapper.
vLLMHigh-throughput OpenAI-compatible inference server. Continuous batching and PagedAttention make concurrent red+blue reasoning under the heartbeat efficient.
BrevNVIDIA GPU cloud where the vLLM/Nemotron endpoint runs. The CF Worker routes both sides' reasoning to it.
Workers AssetsCloudflare deployment method (not Pages) serving the docs and dashboard from the same Worker as the API.
// METRICS

What gets measured

TermDefinition
exfil-success-ratebreaches ÷ attempts against the current policy in a round. The metric that must fall across rounds.
hands-off roundA full round-trip — red breaks in, blue patches, rerun blocks — that runs with no human intervention. At least one is required to prove the loop is recursive.
fail-closedOn HiddenLayer API error or timeout, the action is denied rather than allowed through. The safe default under this track.
escalate-with-approvalOpenShell policy pattern that routes a risky-but-maybe-legitimate action to operator approval instead of a blunt deny. Scores highest in the OpenShell bounty.
0.98 confidenceThe Organized Harness convention: programmatic (unattended) changes only promote once verifier evidence exceeds 98% confidence; below that, human-gated.
// OPERATING RULES

Non-negotiables

  • Docker never guards the judged path. The target→sink egress stays open; OpenShell YAML is the only thing that denies.
  • Enforce at action level. Match on destination + payload shape, never a hardcoded string red can reword.
  • Fail closed. If HiddenLayer errors, deny — never allow through.
  • Never cut the ablation. It is the proof the learning mechanism works, not just the outcome.
  • All Cloudflare deploys use Workers Assets. Not Pages.
